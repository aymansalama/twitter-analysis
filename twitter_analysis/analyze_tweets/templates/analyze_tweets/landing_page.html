Project Description

This project allows user to gather public's opinion or thoughts through the application Twitter while simultaneously perform sentiment analysis on the generated tweets which allows user to analyze the statistics of specified keyword from either positive, negative or neutral.

About Us

This project is for Year 2 Software Engineering Group Project. Started for companies to gather and perform twitter analysis on public's opinion to check their awareness of certain underutilised crops has now been improved to accumulate other user searches. The group consists of Lim Kai Wey, Choo Wei Ken, Chia Bing Shan, Jack Harris, Nurul Hanna Mohd Azhar, Teoh Yee Ting and assisted by our supervisor, Mr Tomas Maul.
